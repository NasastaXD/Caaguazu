# Caaguazú

Portal oficial del departamento de Caaguazú, Paraguay — theme de WordPress más los plugins de módulos de contenido.

- [`caaguazu-theme/`](caaguazu-theme/) — el theme: presentación, templates, Customizer, formularios. Ver [`caaguazu-theme/README.md`](caaguazu-theme/README.md) para instalación completa.
- [`caaguazu-modulos/`](caaguazu-modulos/) — plugin con los módulos de contenido Noticias, Agenda y Ecosistema (CPTs + Customizer de cada uno).
- [`caaguazu-turismo/`](caaguazu-turismo/) — plugin con el módulo Turismo (25 páginas de historia/cultura/gastronomía + integración con los plugins Caaguazú Locales y Caaguazú Portal del ecosistema turístico).
- [`caaguazu-editor-ux/`](caaguazu-editor-ux/) — plugin opcional que simplifica el editor de bloques (Gutenberg) para Entradas: menos bloques, menos ruido visual, vista previa mejorada y un panel editorial propio. No afecta el frontend. Ver [`caaguazu-editor-ux/README.md`](caaguazu-editor-ux/README.md).

Los tres primeros se instalan juntos (theme + 2 plugins); el theme queda funcional sin ellos (nav y home muestran solo lo que el theme trae de por sí — Sobre Caaguazú y Contacto), pero el sitio completo necesita los tres activos. `caaguazu-editor-ux` es independiente de los demás: sólo mejora la experiencia de edición, se puede activar o no sin afectar el resto.
